🍽️ Cozy Restaurant Website
📌 Project Title

Cozy Restaurant Website

A modern restaurant front-end website built using HTML and CSS.
This project showcases a restaurant homepage with a navigation bar, animated food slider, top dishes section, and footer.

✨ Features

🧭 Navigation Bar with:

Logo
Location indicator
Search bar
User login icon
Cart icon
Menu icon
🎞️ Animated Food Image Slider
Displays food images with animation.

🍕 Top Dishes Section

Shows popular food items with star ratings.
📱 Responsive Layout Design

📌 Footer Section with:

About links
Social media icons
Address details
Copyright notice.

🛠️ Technologies Used

HTML5 – Structure of the website
CSS3 – Styling and layout
Font Awesome – Icons
CSS Animation – Image slider animation.

⭐ Future Improvements

Add JavaScript functionality
Add login/signup page
Make fully responsive for mobile
Add food ordering system
